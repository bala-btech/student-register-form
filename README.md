# Student Registration Form

A simple, clean, and responsive HTML-based Student Registration Form that collects student details and displays submitted records in a structured table layout.

## 🚀 Features

* **User Input Fields:** Collects Full Name, Email, Password, and Date of Birth.
* **Gender Selection:** Implements radio buttons for mutually exclusive selection (Male/Female).
* **Skills Checklist:** Checkboxes to select multiple technical skills (HTML, CSS, JavaScript).
* **Form Validation:** Uses native HTML5 `required` attributes to ensure critical data is filled before submission.
* **Data Display Layout:** Includes a pre-styled tabular structure (`<table>`) to present registered student information cleanly.

## 🛠️ Technologies Used

* **HTML5:** For page structure, form elements, and native validations.
